# 2 Fase de aprendizado (Projetos)

Esta pasta marca a segunda fase dos estudos, com projetos um pouco mais completos e organizados.

O objetivo agora é continuar praticando HTML, CSS, JavaScript e Python, mas com mais atenção em:

- layout em grid
- atualização dinâmica no navegador
- feedback visual
- funções mais organizadas no Python
- dados simulados
- projetos com estrutura mais limpa

## Projetos

### 11. Simple Dashboard

Dashboard simples com layout de painel, cards de KPIs, grid de seções, menu superior e dados simulados.

Pasta:

```text
Simple Dashboard/
```

### 12. School Grades System

Sistema de notas escolares com formulário para inserir notas, cálculo automático de média, status aprovado/reprovado e feedback visual por cores.

Pasta:

```text
School Grades System/
```

### 13. Customer List CRUD

Lista de clientes com cadastro, edição, remoção, busca dinâmica, contadores e persistência no navegador.

Pasta:

```text
Customer List CRUD/
```
