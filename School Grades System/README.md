# School Grades System

Projeto 12 da segunda fase de aprendizado.

Este projeto é um sistema simples de notas escolares. O usuário insere as notas de um aluno, a média é calculada automaticamente e a tela mostra se o aluno foi aprovado ou reprovado.

## Tecnologias

- HTML
- CSS
- JavaScript
- Python

## Funcionalidades

- Formulário para inserir nome do aluno e notas
- Cálculo automático da média
- Status aprovado ou reprovado
- Feedback visual com cores
- Barra de desempenho
- Lista de boletins adicionados
- Backend Python com cálculo de média no terminal

## Conceitos praticados

### Frontend

- Formulários
- Eventos de `input`
- Atualização dinâmica com JavaScript
- Manipulação do DOM
- Feedback visual com classes CSS
- Grid responsivo

### Python

- Funções separadas
- Listas
- `for`
- `try/except`
- Validação de entrada
- Dicionários
- Funções retornando dados organizados
